## Problem Statement

Human language contains emotions and opinions that can be analyzed by AI systems. Sentiment analysis is a text recognition technique used to determine whether a piece of text expresses positive, negative, or neutral sentiment.

## Project Task

- Use a pre-trained AI library.
- Perform text recognition on user input.
- Interpret and display model outputs.
- Understand the fundamentals of Natural Language Processing.

----

## Methodology
Step 1: User Input

The user enters a sentence or paragraph.

Example:

"I enjoy learning Artificial Intelligence."

Step 2: Text Processing

The input text is passed to a pre-trained sentiment analysis model provided by the TextBlob library.

Step 3: Sentiment Analysis

The model evaluates the text and produces a sentiment polarity score ranging from:

-1.0 = Very Negative
0.0 = Neutral
+1.0 = Very Positive
Step 4: Classification

Based on the polarity score, the system classifies the text as:

Positive
Negative
Neutral
Step 5: Output Display

The sentiment result and score are displayed to the user.
